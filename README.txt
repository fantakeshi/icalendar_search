ical形式のカレンダーを多言語(主に日本語)で検索するためのPythonモジュールです。
Googleカレンダーの日本語での検索が信用できないので作りました。
MITライセンスで使用可能です。

・使用方法
  src/example.py を見てください

・インストール方法
  解凍したディレクトリで

  > sudo python setup.py install

  と入力してください。Windowsの場合は

  > python setup.py install

  です

・リリース履歴
  2011/06/10 First Release
